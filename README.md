# Pissa sales dashboard

Performed exploratory data analysis on pizza sales data to quantify total sales, identify top- and bottom-performing products, and generate actionable insights to support data-driven business decisions.

- Microsoft Excel 
- PowerBI (Dax Calculations, KPIs, Visualizations) 
- Canva (for the design)
